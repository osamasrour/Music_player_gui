## About
An application used to play music that uses a GUI (pygame, tkinter, mutagen)

## How To Install The Modules We Need

```bash
pip install pygame==2.6.0
```
```bash
pip install tkintertable==1.3.3
```
```bash
pip install mutagen==1.47.0
```
